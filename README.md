# LinkedIn Clone

A LinkedIn-inspired social networking application built to practice modern frontend development with React, Redux, and Firebase.  
The project focuses on authentication, global state management, and component-based UI design.

---

## Live demo
https://linkedin-clone-ap1wx1r4i-praise-wawerus-projects.vercel.app/

## Features

- User authentication (login & logout) using Firebase Authentication
- Persistent user state with Redux
- Real-time authentication state tracking
- Feed layout similar to LinkedIn
- Responsive UI components
- Logout functionality with Firebase sign-out

---

## Tech Stack

- **React** – Component-based UI
- **Redux Toolkit** – Global state management
- **Firebase Authentication** – User login/logout
- **Firebase** – Backend services
- **Material UI Icons** – UI icons
- **CSS** – Styling
